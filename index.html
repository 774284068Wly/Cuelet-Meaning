02 / WORKFLOW

三步开始解读

扩展不会自带密钥。你需要配置一个支持图片输入的 OpenAI 兼容视觉模型接口。

1

加载扩展

解压 ZIP，在 chrome://extensions 打开开发者模式并加载文件夹。

2

配置模型

在设置页填写 Base URL、API Key 与支持图片输入的模型名称。

3

选择图片

点击“选图解读含义”，或在图片右键菜单中直接开始分析。
